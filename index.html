About

This is a blatant attempt to piggyback off the popularity of the global Hacktoberfest event from Digital Ocean and GitHub. Submit four pull requests to qualifying devICT projects during the month of October and receive a t-shirt!

Hacktoberfest is a great way for anyone to learn about open source, contributions to a team, and helps build a better resume and portfolio.
This event was created to:

  • Bring awareness to the open source projects that make Wichita a better place to live
  • Bring light to the diversity of the Wichita Tech Commnity
  • Create opportunities for those individuals who are underrepresented in tech

Getting Started

Pick an Issue

Look through the project issues and choose one or suggest a change!

Make an Improvement

Use Git and GitHub to make your change. See Kati Michel's helpful tutorial, if needed.

Submit a Pull Request

Submit a pull request with your proposed change.

Enjoy Your T-Shirt

After four pull requests, you are eligible to receive a free t-shirt!

Qualifying Projects

We will count pull requests opened against any of the Wichita developer community organizations listed below, including the repo for this site! We will also count pull requests opened against the individual project foodtrucksnear.me. Would you like your organization or project to show up on these lists? Just let us know! File an issue against this site's repo or maybe make a pull request.
